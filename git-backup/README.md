# Git Config Backup Add-on for Home Assistant

Automatically backup your Home Assistant configuration to a Git repository (GitHub, GitLab, Bitbucket, etc.).

## Features

- Automatic scheduled backups
- Version history with meaningful diffs
- Support for HTTPS and SSH authentication
- Smart defaults exclude databases and logs
- Works with GitHub, GitLab, Bitbucket, and self-hosted Git

## Quick Start

1. Add this repository to your Home Assistant add-on store
2. Install the "Git Config Backup" add-on
3. Configure your Git repository URL and credentials
4. Start the add-on

See [DOCS.md](DOCS.md) for detailed configuration options.

## Installation

Add this repository URL to your Home Assistant add-on store:

```
https://github.com/sttts/ha-git-backup
```
